# Health Directory Frontend

A comprehensive healthcare directory for Malaysia, built with React, TypeScript, and Tailwind CSS.
This project allows users to search for hospitals and doctors across various specialities.

## Features

-   **Hospital Directory**: Browse and search hospitals.
-   **Doctor Directory**: Find specialists by name or hospital.
-   **Responsive Design**: Optimized for mobile and desktop.
-   **Search & Filtering**: Easy to find what you need.

## Tech Stack

-   React
-   TypeScript
-   Vite
-   Tailwind CSS
-   shadcn/ui

## Getting Started

### Prerequisites

-   Node.js (v18 or later)
-   pnpm (recommended) or npm

### Installation

1.  Clone the repository
2.  Install dependencies:

```bash
cd frontend
pnpm install
```

3.  Start the development server:

```bash
pnpm dev
```
