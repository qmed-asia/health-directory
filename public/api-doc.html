Health Directory API Documentation

Welcome to the Health Directory API. This API provides access to scraped data of hospitals and doctors in Malaysia.

Base URL

/api

Endpoints

GET /hospitals

Returns a list of hospitals.

Response Schema

Field Type Description
"" (Empty Key) String Name of the Hospital (e.g., "KPJ Batu Pahat Specialist Hospital")
HOSPITAL GROUP String Group name (e.g., "KPJ")
ADDRESS String Full address
CITY String City
STATE String State
NO. TELEPHONE String Contact number
Website String Official website URL
Doctor Directory String Link to the hospital's doctor directory page
GET /doctors

Returns an aggregated list of all doctors from all available sources.

Note: Since data is scraped from different sources, the schema varies.
GET /doctors/:source

Returns a list of doctors for a specific source/hospital group.

Available Sources: columbia, gleaneagles, tungshin, georgetownspecialist
